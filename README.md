# Train-Scheduling

This app is a train schedule simulator.

## How to Use

1. Input the train name, destination, frequency, and the first train time
2. Your new train will be added to the table with the next arrival automatically calculated

## Technologies Used

- Firebase
- JQuery
- Javascript
- CSS
- HTML
